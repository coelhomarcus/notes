# REST API

Usada para desenvolvimento web.

## Query Parameters

- URL -> minhaapi.com
- EndPoint -> nomes
- As querys são: letras = 10 e letrainicial = m

```
https://minhaapi.com/nomes?letras=10&letrainicial=m
```

## Path Parameters

- URL -> minhaapi.com
- EndPoint -> nomes
- Os parametro é o 0 que pode ser um id

```
https://minhaapi.com/nomes/0
```
